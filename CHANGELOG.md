# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0-beta.1] - 2026-01-12
### Added
- Health check endpoint `/health`.
- Clear Chat functionality.
- Copy to clipboard for AI responses.
- Version badge in the UI.
- MIT License and Contributing guidelines.
- GitHub Actions CI workflow.
- Unit and Integration tests.
- Logging system in backend.
- Favicon and meta description.

### Fixed
- CSS compatibility issues for `background-clip` and `line-clamp`.
- Improved model selection fallback logic.
- Smoother transitions and responsive layouts.
